There are fundamentally two types of control flow structures in Swift. Loops and Conditional Statements. Loops are handled by using 'for', 'while' and 'do while'. Conditional Statements are handled using 'if else' and 'switch'.

There is some more terminology that goes along with each type of control flow. This chapter tries to explain the way loops are handled in Swift Programming Language.

A loop is a way of executing a task multiple times. Usually there is a condition that needs to be satisfied to repeat the loop. The loop execution stops when the condition is not met to repeat the loop, or if there is a use of 'continue' or 'break' keywords to break the flow of execution.
